# DanMeet  
2022년도, 2학기가 되며 본격적으로 대면체제로 돌입됐다. 그러나 당신..이런 상황이 아닌가?  
<br/>  

*"등교했는데, 저 혼자 아는 사람이 없어요.. 저도 동기들이랑 놀고 싶은데 어떻게 친해지죠?"*  
*"선배들이랑 친해지고 이것저것 물어보고 싶은데, 어떻게 해야할지 모르겠어요 ㅠㅠ"*  
<br/>  

이런 당신을 위한, 매칭&채팅 서비스. 익명을 통해 부담없이 같은 대학에서 친해지고 싶은 사람을 만나보세요.  
<br/>
